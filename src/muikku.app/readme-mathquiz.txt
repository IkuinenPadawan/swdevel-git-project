MathQuiz.java is a simple MathQuiz.
Running it requires a java jdk installation.

1. Open command line (or a terminal) and navigate to the root directory of MathQuiz.java
2. Write on command line: javac MathQuiz.java
3. Then: java MathQuiz
4. When you want to quit, type 'exit'. (ctrl + c also works, as always).
