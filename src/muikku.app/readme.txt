Calculator.java is a simple calculator app.
Running it requires a java jdk installation.

1. Open command line (or a terminal) and navigate to the root directory of Calculator.java
2. Write on command line: javac Calculator.java
3. Then: java Calculator
4. When you want to quit calculating, type exit. (ctrl + c also works, as always).
5. Additional commands: 'avg', 'sum'.
	-'avg': prints average of all previous results in that session.
	-'sum': prints sum of all previous results in that session.
